# Building with Google Antigravity

이 튜토리얼은 구글의 에이전트형 IDE인 'Antigravity'를 활용하여 단순한 웹 정보 추출부터 복잡한 풀스택 애플리케이션 및 유닛 테스트 생성까지의 과정을 안내합니다.

* Antigravity는 사용자의 지침을 이해하고 브라우저 제어, 코드 작성, 테스트 및 실행을 자율적으로 수행하는 에이전트 중심 플랫폼입니다.
* 웹 뉴스 하이라이트 추출 사례를 통해 에이전트가 브라우저를 직접 구동하고 사이트를 탐색하며 정보를 요약하는 과정을 보여줍니다.
* Python과 Flask 프레임워크를 기반으로 1일 기술 컨퍼런스 안내 사이트를 생성하며, 데이터 구조 정의부터 검색 기능 구현까지 전체 스택을 자동 구축합니다.
* 사용자는 Agent Manager에서 추상적인 명령을 내리고, Editor 모드로 전환하여 실시간으로 코드를 수정하거나 추가 기능을 에이전트에게 요청할 수 있습니다.
* 뽀모도로 타이머와 같은 생산성 앱 제작 시, 에이전트가 디자인 스타일을 조정하고 필요한 이미지 에셋을 생성하며 검증 영상을 포함한 미디어 아티팩트를 제공합니다.
* 에이전트의 사고 과정(Thought Process)과 구현 계획(Implementation Plan)을 가시화하여 개발자가 에이전트의 작업 단계를 상세히 모니터링할 수 있도록 지원합니다.
